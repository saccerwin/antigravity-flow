# iOS Project Analysis & Audit

Use this skill for periodic reviews, security audits, or architectural assessments.

## Analysis Domains

### 1. General Architecture
- Audit codebase against `ios-app-framework-skills` and `swiftui-patterns`.
- Check for dependency bloat and proper modularization.

### 2. Localization & Accessibility
- Check for hardcoded strings using `ios-localization`.
- Audit VoiceOver and Dynamic Type support via `ios-accessibility`.

### 3. Security Audit
- Use `ios-security` and `device-integrity`.
- Check Keychain usage, ATS configuration, and Privacy Manifests.

### 4. On-Device Intelligence & Hardware
- If using ML, check `apple-on-device-ai` and `coreml`.
- For hardware, reference `ios-hardware-skills` (BLE, NFC, Motion).

## Reporting
- Generate reports in `docs/superpowers/analysis/`.
