# Kỹ năng Dữ liệu & Phân tích

> **Dành cho Data Analysts, Data Engineers, BI Analysts và Data Scientists.** Viết SQL, khám phá datasets, tạo insights, xây dựng visualizations và biến dữ liệu thô thành câu chuyện rõ ràng.

## Kỹ năng con

| Kỹ năng | Mục đích | Kích hoạt khi |
|---------|----------|---------------|
| `viết-truy-vấn` | Viết và tối ưu SQL queries | "viết query cho", "SQL để tìm" |
| `khám-phá-dữ-liệu` | Profile datasets, tìm patterns, kiểm tra chất lượng | "khám phá dữ liệu", "bảng này chứa gì" |
| `tạo-insights` | Phân tích thống kê, nhận diện xu hướng | "phân tích", "tìm insights trong" |
| `tạo-dashboard` | Dashboard HTML tương tác với Chart.js | "tạo dashboard", "trực quan hóa" |
| `kiểm-tra-dữ-liệu` | Kiểm tra chất lượng, validation schema, phát hiện bất thường | "kiểm tra dữ liệu", "validate data" |

## Ví dụ sử dụng

```
"Tạo dashboard theo dõi doanh thu theo tháng, chia theo kênh bán hàng"
→ Agent tạo file HTML tự chứa với Chart.js, dropdown filters cho kênh và thời gian, bảng chi tiết
```

## Workflow liên quan

- `/data-write-query` — Sinh SQL tối ưu từ ngôn ngữ tự nhiên
- `/data-explore-data` — Profile và tóm tắt dataset
- `/data-analyze` — Phân tích thống kê với phát hiện chính
- `/data-create-viz` — Tạo biểu đồ hoặc visualization
- `/data-build-dashboard` — Dashboard HTML tự chứa
- `/data-validate` — Kiểm tra chất lượng dữ liệu
