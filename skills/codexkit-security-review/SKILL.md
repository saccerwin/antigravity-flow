# codexkit-security-review

Review changes through an OWASP and CWE lens. Check auth, authorization, secrets handling, input validation, injection risk, unsafe deserialization, crypto misuse, and supply-chain exposure. Treat trust-boundary mistakes and missing tests as first-class security findings.
