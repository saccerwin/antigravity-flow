# iOS Brainstorming Workflow

Use this skill as the starting point for any creative or architectural task in your iOS app.

## Process

1. **Invoke Standard Brainstorming:**
   - Use the `brainstorming` skill to explore the "What" and "Why".
   
2. **Apple Platform Sourcing:**
   - Ask: "How does this align with Apple's Human Interface Guidelines (HIG)?"
   - Cross-reference `ios-platform-skills`.

3. **Technical Viability:**
   - Check against `ios-app-framework-skills` and `swift-core-skills`.
   
4. **Draft Spec:**
   - Save the result to `docs/superpowers/specs/`.
