# x-devs

**Social media skill** — Build a developer following on Twitter/X.

## What It Does

Guides you through X/Twitter for developers: thread structure, code screenshots, engagement timing, hashtags, building following, and avoiding cringe content.

## When to Use

- Writing technical threads
- Building a developer audience on X
- Understanding what content works for devs
- When you say "Twitter," "X," "technical threads," or "dev Twitter"

## Quick Start

```
/x-devs

Help me turn this blog post into a Twitter thread.
It's about error handling patterns in Rust and I
want to maximize engagement from the Rust community.
```

## Key Sections

1. **Understanding Dev Twitter** — Who's there, what works
2. **Thread Structure** — Hook, meat, CTA
3. **Code Screenshots** — Tools and best practices
4. **Engagement Timing** — When to post
5. **Building Following** — The flywheel
6. **Avoiding Cringe** — What fails

## Thread Structure

```
Tweet 1: HOOK (compelling, no links)
Tweet 2: Context/Promise
Tweets 3-8: Key points with examples
Tweet 9: Summary/TL;DR
Tweet 10: CTA + engagement ask
```

## What Works vs What Fails

| Works | Fails |
|-------|-------|
| Technical threads | Humble brags |
| Code screenshots | Plain links |
| Building in public | Engagement bait |
| Real opinions | Generic motivation |
| Authentic stories | AI-generated slop |
