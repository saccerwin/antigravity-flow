# codexkit-bug-hunt

Use this when something is broken and you need disciplined debugging instead of guesswork. Capture expected versus observed behavior, reproduce the issue with the cheapest reliable path, narrow the failure boundary, patch the smallest credible cause, and verify the original failure plus nearby regressions.
