# codexkit-automation-designer

Design recurring automations that are self-contained, low-noise, and easy to reason about. Define the recurring problem, write a prompt that describes only the task, specify the output shape and success criteria, add explicit skip conditions, and choose a schedule that matches the intended response cadence.
