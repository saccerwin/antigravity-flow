# codexkit-productivity-ops

Use this skill for operator productivity work: prioritizing competing requests, triaging inboxes, building follow-up loops, and restoring execution rhythm when work feels scattered. Favor practical structures such as now/next/later queues, meeting prep checklists, and dependency-aware follow-up plans.

## Capabilities

- priority-triage
- follow-up-system-design
- operator-rhythm-planning




## Expected Output

- Turn a messy request list into a prioritized queue with a small next action for each item.
- Separate must-do work, delegated work, and parked work instead of flattening everything into one list.
- Bias toward a realistic operator cadence that can be followed daily or weekly.
