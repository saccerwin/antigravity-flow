# codexkit-engineering-delivery

Use this skill when engineering work needs coordination rather than pure coding: bug triage, release readiness, stakeholder handoffs, and post-incident follow-up. The output should help execution move forward with fewer hidden assumptions.

## Capabilities

- bug-triage
- release-handoff
- incident-followup




## Expected Output

- Summarize engineering work in terms of owner, status, risk, and unblocker.
- Separate fix work, verification work, and communication work.
- Prefer operational clarity over technical theater.
