# CodexKit iOS Performance

Use this skill for targeted performance work in production-oriented SwiftUI apps.

## Workflow
1. Define the symptom clearly and reproduce under realistic conditions.
2. Measure before optimizing.
3. Trace hot paths through View composition, ViewModel updates, async work, image loading, persistence, and networking.
4. Fix the smallest high-impact bottleneck first.
5. Re-measure and confirm UX remains correct.
