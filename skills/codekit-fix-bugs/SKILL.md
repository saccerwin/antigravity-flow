# CodexKit Fix Bugs

Use this skill for reliable bug-fix execution in SwiftUI-first apps.

## Workflow
1. Lock the failing scenario and impacted screens.
2. Prove root cause before broad edits.
3. Make the smallest safe change that resolves the defect.
4. Recheck neighboring flows, state transitions, and async side effects.
5. Add or tighten regression coverage.
6. Summarize blast radius and residual risks.
