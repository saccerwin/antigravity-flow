# youtube-devrel

**Video content skill** — Create technical YouTube content and screencasts.

## What It Does

Guides you through developer YouTube: tutorial structure, screen recording setup, live coding, shorts vs long-form, thumbnails, and SEO for technical content.

## When to Use

- Creating technical video tutorials
- Setting up screen recording workflow
- Planning YouTube content strategy
- When you say "YouTube," "screencast," "video tutorial," or "live coding"

## Quick Start

```
/youtube-devrel

I want to create a tutorial video showing how to
build a CLI tool in Go. Help me plan the structure,
what equipment I need, and how to optimize for search.
```

## Key Sections

1. **Understanding Developer YouTube** — Audiences, content types
2. **Tutorial Structure** — Opening hook, pacing, chapters
3. **Screen Recording Setup** — Equipment, settings, workflow
4. **Live Coding Content** — Formats, best practices
5. **Shorts vs Long-Form** — When to use each
6. **Thumbnails** — Design principles for devs
7. **YouTube SEO** — Keywords, titles, descriptions

## Tutorial Structure

```
0:00 - Hook (show end result)
0:30 - Prerequisites & setup
2:00 - Concept explanation (why)
4:00 - Step 1: [First major step]
...
XX:00 - Final result demo
XX:30 - Recap & CTA
```

## Shorts vs Long-Form

| Long-form | Shorts |
|-----------|--------|
| Search traffic | Discovery traffic |
| Watch time revenue | Low revenue |
| Depth | Breadth |
| 15-45 min | < 60 seconds |
