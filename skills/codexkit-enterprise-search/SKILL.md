# codexkit-enterprise-search

Use this skill when information is spread across documents, notes, playbooks, tickets, or internal references. Focus on search strategy, evidence grouping, and concise answer synthesis rather than dumping every matching source. The goal is to help an operator or analyst get to a trustworthy answer quickly.

## Capabilities

- search-strategy-design
- fragmented-context-synthesis
- answer-compression




## Expected Output

- State what was found, what is still missing, and the best next search path.
- Compress large doc sets into a short answer plus cited supporting facts.
- Avoid pretending retrieval is complete when the evidence is thin.

This skill bundles pack references under `references/`. Load only the files needed for the current task.
