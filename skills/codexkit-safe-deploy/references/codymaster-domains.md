# CodyMaster Domain Map

This pack curates the open CodyMaster skill catalog into CodexKit-native runtime assets.

Domains included:
- Engineering: test-first development, debugging, quality gates, code review
- Operations: deploy safety, identity guardrails, worktree isolation, secrets hygiene
- Product: planning, UX, UI preview, documentation, project bootstrap, JTBD
- Growth: content factory, ads tracking, CRO methodology
- Orchestration: continuity, skill chains, discovery, deep search, dashboard status

Use this document as a routing hint, not as a source of truth for implementation details.
