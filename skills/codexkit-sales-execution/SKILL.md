# codexkit-sales-execution

Use this skill for real sales execution work: preparing discovery calls, briefing account context, organizing objections, and drafting focused follow-ups. Outputs should help a rep move a deal forward with clearer context and cleaner next steps.

## Capabilities

- account-briefing
- discovery-prep
- follow-up-drafting




## Expected Output

- Turn sparse account context into a short brief with likely pains, goals, and open questions.
- Separate qualification gaps from next-step messaging.
- Bias toward concrete follow-up actions instead of generic persuasion tips.
