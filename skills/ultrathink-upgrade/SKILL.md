# codexkit-ultrathink-upgrade

Use UltraThink-style upgrades when routing quality, confidence handling, or workflow discipline needs to improve without replacing the whole runtime.

## Operating Notes
- Prefer weighted routing over flat keyword matching.
- Use confidence bands and top-1 vs top-2 margin to decide route, downgrade, or clarify.
- Add workflow-specific memory read and write policy rather than global auto-capture.
- Measure route quality before and after every major routing change.
