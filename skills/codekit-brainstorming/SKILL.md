# CodexKit Brainstorming

Use this skill for structured option analysis before implementation.

## Workflow
1. Recall durable preferences from memory before final recommendation.
2. Frame the user problem, constraints, and success metrics.
3. Generate two to four realistic implementation options.
4. Evaluate complexity, risk, testability, and delivery speed.
5. Recommend one path with fallback and rollout notes.
